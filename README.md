# mini-projects

A collection of small frontend projects built while learning web development.

---

## Projects

### Interactive Splitting Circles
An interactive, responsive UI grid canvas built to study recursive layout partitioning, fine-vector geometry, and visual style inheritance. Inspired by the minimal, geometric design principles of the Linear Method.
* **Directory:** [`/interactive-splitting-squares`](./interactive-splitting-squares)
* **Stack:** React · Vite

![Interactive Splitting Circles Demo](./interactive-splitting-squares/demo.gif)

### Interactive Sleep Notebook
A single-page React app tracking the architectural waves of human sleep. Features real-time state readouts, scratch-reveal components, and an organic theme layout engine.
* **Directory:** [`/interactive-notebook`](./interactive-notebook)
* **Stack:** React · Vite

### Dark / Light Toggle
An animated sun icon that transitions between dark and light modes, with theme persistence saved via browser localStorage.
* **Directory:** [`/dark_light_toggle`](./dark_light_toggle)
* **Stack:** HTML · CSS · JavaScript

### Spinning Clock Animation
A digital clock featuring three concentric SVG progress rings tracking hours, minutes, and seconds dynamically.
* **Directory:** [`/'spinning_clock_animation '/`](./spinning_clock_animation%20)
* **Stack:** HTML · CSS · JavaScript

### Tic Tac Toe v1
A foundational tic tac toe game using linear conditional logic to run a basic blocking AI script.
* **Directory:** [`/tic_tac_toe_v1`](./tic_tac_toe_v1)
* **Stack:** HTML · CSS · JavaScript

### Tic Tac Toe v2
A rebuilt engine running a minimax calculation algorithm for optimal AI moves, paired with a cleaner UI.
* **Directory:** [`/tic_tac_toe_v2`](./tic_tac_toe_v2)
* **Stack:** HTML · CSS · JavaScript

### 100 Days CSS
Daily layout exercises and style sprints to test CSS box-model fundamentals. Concluded at Day 8.
* **Directory:** [`/100_days_css`](./100_days_css)
* **Stack:** HTML · CSS
