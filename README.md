# mini-projects


A collection of small frontend projects built while learning and teaching web development.

## Projects

### 🌗 Dark / Light Toggle
An animated sun icon that toggles between dark and light mode. Theme persists via localStorage.
* **Stack:** HTML · CSS · JavaScript


### 🕐 Spinning Clock Animation
A digital clock with three concentric SVG progress rings — one each for hours, minutes, and seconds.
* **Stack:** HTML · CSS · JavaScript


### ❌ Tic Tac Toe v1
A working tic tac toe game with a basic blocking AI (if's and else's). 
* **Stack:** HTML · CSS · JavaScript


### ❌ Tic Tac Toe v2
Rebuilt version with a minimax AI that plays optimally. Cleaner UI and logic.
* **Stack:** HTML · CSS · JavaScript


### 📓 Interactive Sleep Notebook
A single-page, responsive React app tracking the architectural waves of human sleep. Features real-time state readouts, scratch-reveal components, and a custom light, dark, and organic theme engine.
* **Stack:** React · Vite


### 🎨 100 Days CSS
Daily CSS challenge exercises and experiments. Stop at day 8.
* **Stack:** HTML · CSS
